Страничка статистики существует как отдельно для каждого региона каталога (СПб, Москва, Белград, ) так и для общая. Попасть на неё можно по ссылке с главной страницы или нажав соответствующую кнопку на странице с картами.

Важный нюанс - эта статистика не включает большинство карт, расположенных в разделе "Старты на карте".

Диаграммы

Первая часть страницы содержит три статистические диаграммы с общей информацией о картах каталога:

  1. Количество карт по типам (включая рогейнерские)
  2. Площадь спортивных карт
  3. Количество карт по годам
Диаграммы строятся автоматически по данным каталога.

Дополнительно показаны самая старая и самая большая карты в каталоге, высчитывается общая (приблизительная) площадь всех карт каталога.

Площадь большинства карт определяется, исходя из координат листа. Для компенсации того, что реальная карта практически всегда меньше, введён понижающий коэффициент 70%. Площадь некоторых (в основном, крупных) карт указана вручную.

Таблица рейтинга авторов

Во второй части у нас рейтинговая таблица авторов-составителей карт. Для каждого автора указано количество карт, в составлении которых он принимал участие, их оценочная площадь и дана ссылка на страницу O-maps, на которой будут показаны исключительно карты этого автора.

Если у карты указано несколько авторов, то и её площадь, попадающая в рейтинг, будет разделена между всеми участниками.

Также как и диаграммы, рейтинговая таблица высчитывается автоматически..
